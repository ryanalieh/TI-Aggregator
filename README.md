# TI Aggregator — ti.lxnar.dev

Threat Intelligence aggregator. Paste an IP, domain, URL, or file hash — get a unified report from all sources in parallel.

## Sources

| Source | Types | Key needed |
|--------|-------|-----------|
| VirusTotal | IP, domain, URL, hash | Yes (free tier) |
| AbuseIPDB | IP | Yes (free tier) |
| URLScan.io | URL, domain | Yes (free tier) |
| DNS (Google DoH) | Domain, URL | No |
| SSL (crt.sh) | Domain, URL | No |
| WHOIS (RDAP) | Domain | No |
